# OBhoD_BLOK

Android-клиент для VPN-туннеля WireGuard, который подключается через TURN-ретранслятор VK для обхода блокировок. Снаружи трафик выглядит как зашифрованный медиапоток звонка WebRTC, а не как классический VPN.

**Пакет:** `net.qwdtt.client` · **Версия:** 1.3.4  
**Telegram-бот для получения конфигов:** [@obhod_int_bot](https://t.me/obhod_int_bot)

## ⚡ Особенности OBhoD_BLOK

| Функция | Описание |

| **Профили** | Поддержка нескольких конфигов, папки, перетаскивание, пинг |

| **Исключения** | Белые и чёрные списки приложений для выборочного обхода |

---

## 🛡️ Как это работает

Трафик маскируется под обычные аудиозвонки VK (WebRTC):
1. Клиент получает TURN-учётные данные из VK;
2. Поднимает DTLS через TURN relay;
3. Пакеты обёрнуты в RTP и шифруются WRAP (ChaCha20-Poly1305);
4. Системные фильтры операторов (DPI) видят стандартный звонок ВКонтакте.
